SpringBoot中如何跨域

发表于2020-07-09|更新于2020-07-09
|字数总计:240|阅读时长:1分钟|阅读量:

在SpringBoot中，可以继承WebMvcConfigurerAdapter，重写addCorsMappings方法

后端代码：

@Configuration
public  class WebMvcConfg extends WebMvcConfigurerAdapter {
@Override
public void addCorsMappings(CorsRegistry registry) {
    //设置允许跨域的路径
    registry.addMapping("/**")
            //设置允许跨域请求的域名
            //当**Credentials为true时，**Origin不能为星号，需为具体的ip地址【如果接口不带cookie,ip无需设成具体ip】
            .allowedOrigins("http://127.0.0.1:5500")
            .allowedHeaders("*")
            //是否允许证书 不再默认开启
            .allowCredentials(true)
            //设置允许的方法
            .allowedMethods("*")
            //跨域允许时间
            .maxAge(3600);
}

}

前端代码

 $.ajax({
        type: "post",
        url: "http://localhost:9000/user/login",
        data: JSON.stringify(json),
        dataType: "json",
        contentType: "application/json",
        success: function(data){
            console.log(1);
        },
      });

注意:如果后端在接受参数时用的是@ResponeBody，在AJAX中需要设置contentType为：”application/json”，否则会出现415错误，另外，我们需要将前端得json对象通过JSON.stringify()得方式转换为字符串对象，否则会报400错误
